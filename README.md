# TLSProfiler

Behavioral Profiling via TLS Handshakes and Flow Dynamics

## Overview
TLSProfiler is a Python-based network traffic analysis tool that analyzes encrypted TLS traffic without decrypting payloads.

## Objectives
- Read PCAP files
- Extract packet and TLS metadata
- Generate JA3 fingerprints
- Perform traffic classification
- Detect anomalous behavior
- Visualize results using Streamlit

## Tech Stack
- Python
- Scapy
- PyShark
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Wireshark
- tcpdump
